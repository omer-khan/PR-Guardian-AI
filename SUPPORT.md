# **Support for PR Guardian AI**

Thank you for using **PR Guardian AI**.

If you need help, encounter a bug, or want to request a feature, please use the project's GitHub Issues page.

### **How to Get Support**

1. Go to the Issues page:
   [https://github.com/AmirhosseinHonardoust/PR-Guardian-AI/issues](https://github.com/AmirhosseinHonardoust/PR-Guardian-AI/issues)

2. Click **New Issue**

3. Select the appropriate template (Bug Report / Feature Request)

4. Provide:

   * A clear description of the problem
   * Screenshots (optional)
   * Steps to reproduce
   * Repository link (if relevant)
